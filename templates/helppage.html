Admin: Managing Users

You can click on any user's name from the manage users page to edit that user's profile. From the modify user page, you can change a user's personal details, resident status, etc. as well as change his/her password. To prevent permanent lockout, if you are the only superadmin, you will be prevented from changing your own superadmin status. Everything else will be saved.

It is highly recommended that users not be deleted. You should instead deactivate them. But if you really need to delete a user, you can do so from the Django admin pages.
